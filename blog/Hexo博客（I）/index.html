Ubuntu网络无法连接

  1. 删除NetworkManager缓存文件
  1. 修改/etc/NetworkManager/NetworkManager.conf
  1. 重启NetworkManager

Ubuntu没有node和npm

  1. 使用sudo apt install nodejs npm
  2. 注意值安装node不会自带npm
  3. 安装的版本低，更新麻烦，放弃了

VMware

VMware开机关机的时候会使用麦克风

Hexo

使用不支持的语法hexo g将报错，例如[dco], {#id}

同时默认不支持LaTeX语法，使用主题可能可以使用，有待实验。
